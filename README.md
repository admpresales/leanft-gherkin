# uftdev-aos-web-gherkin
A gherkin test designed to run in UFT Developer (Formerly LeanFT) and prepared for upload to ALM Octane. 

## Overview
IntelliJ - maven project which uses UFT Developer and gherkins features and files and cucumber for execution. This executes a test against AOS
